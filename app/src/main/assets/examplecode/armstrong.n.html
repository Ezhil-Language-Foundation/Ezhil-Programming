நிரல்பாகம் அர்ம்ஸ்ற்ரோங்( எண்)
    இடய்= எண்
    மொத்தம் = 0
    @( இடய் > 0 ) வரை 
       மொத்தம் = மொத்தம் + (இடய்%10)^3
       இடய் = int(இடய்/10)
    முடி
    @( மொத்தம் == எண் ) ஆனால்
        பதிப்பி "எண் ",எண்," ஒரு அர்ம்ஸ்ற்ரோங் எண்"
        பின்கொடு True
    முடி
    #பதிப்பி "எண் ",எண்," ஒரு அர்ம்ஸ்ற்ரோங் கிடபயத்து"
    பின்கொடு False
முடி
