நிரல்பாகம் செயற்கை_neuron( input, wts, t)
    
    L = நீளம்(input)
    @( L != நீளம்(wts)) ஆனால்
           பதிப்பி "different sizes"
           பின்கொடு False
    முடி
    
    சமம்= 0.0
    itr = 0
    @( itr < L ) வரை
        சமம்= சமம் + input[itr]*wts[itr]
        itr = itr + 1 #don't introduce an infinite loop!
    முடி
    
    @( சமம் >= t ) ஆனால்
       பின்கொடு True
    முடி
    பின்கொடு False
முடி
