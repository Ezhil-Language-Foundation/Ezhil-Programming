# 5 இல் தொடங்கி 1.5 படிகளால்  வரை உள்ள geometric எண் தொடர்
தொடக்கம்=5; படி=1.5; 
எண்_தொடர் = range(5,61+1,7)

 @( டி = 1, டி < len(எண்_தொடர்), டி = டி+1 )  ஆக
       எண்_தொடர் [ டி ] = எண்_தொடர் [ டி - 1 ]* படி
 முடி

# கூட்டு_சூத்திரம்
அளவெண்=len(  எண்_தொடர் )
கூட்டு_சூத்திரம் = எண்_தொடர்[0]*(1.0 - படி^(அளவெண்))/(1.0 - படி)

பதிப்பி "geometric எண் தொடர்", எண்_தொடர்
பதிப்பி "geometric கூட்டு =>",  கூட்டு (  எண்_தொடர் ) 
பதிப்பி "geometric  கூட்டு சூத்திரம் => ", கூட்டு_சூத்திரம்
assert( கூட்டு (  எண்_தொடர் )  == கூட்டு_சூத்திரம் )
